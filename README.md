# Syna 💜
### Finanças do casal, juntos.

> Um app pensado para casais que querem organizar o dinheiro de forma simples, visual e sem estresse.

🔗 **[Ver site ao vivo](https://madsystemics.github.io/Syna)**

---

## ✨ Funcionalidades

- 💸 Controle de gastos compartilhado
- 📊 Visão clara das finanças do casal
- 🔐 Login individual com conta segura
- 📱 Design responsivo para mobile e desktop
